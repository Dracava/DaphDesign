Home Budgeting Dashboard

Project Overview

A prototype of a smart home budgeting dashboard, specifically created for elderly who struggle with tiny interfaces on smartphones. This project focuses on accessibility and user-friendly design principles to ensure that financial management is accessible to all users, regardless of their technical comfort level.

Key Features

  • Large, readable text and buttons for easy interaction
  • Intuitive navigation system with clear visual hierarchy
  • Accessibility-focused design with high contrast and clear typography
  • Simple, uncluttered interface to reduce cognitive load
  • Touch-friendly button sizes for mobile devices

Technologies Used

  • Figma for prototyping and design
  • User research and testing methodologies
  • Accessibility guidelines implementation (WCAG)
  • Responsive design principles

Design Challenges

Creating an interface that balances simplicity with functionality while ensuring accessibility for elderly users required careful consideration of typography, color contrast, and interaction patterns. The challenge was to make complex financial data understandable and manageable for users who may not be comfortable with modern digital interfaces.
