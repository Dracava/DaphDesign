PromptlyUX

A personally designed and developed AI-Driven Figma plugin for designers. Created for my master thesis at Leiden University.

Project Overview

PromptlyUX is a Figma plugin designed to support UI/UX designers in crafting effective prompts for AI tools like ChatGPT. It bridges the gap between creative workflows and AI systems by offering structured prompt guidance tailored to the design process. This project was developed as part of my master's thesis on AI-assisted design workflows.

Key Features

  • AI Chat Interface with contextual design guidance
  • Prompt Guidance System with 10-question survey
  • Prompt Library with categorized examples and UX keywords
  • Interactive UI Tabs for different design stages (Personas, Style Guide, etc.)
  • Accessibility Tools with adjustable text sizes
  • Saved Prompts management for reuse and editing
  • Prompt Generator for instant variations

Technologies Used

  • Figma Plugin API for integration
  • React for plugin interface development
  • Large Language Model Integration
  • User Research & Testing methodologies
  • JavaScript for plugin functionality
  • CSS for responsive plugin design

Research & Development

This project explores how structured prompting can enhance creativity, efficiency, and AI literacy among designers. The challenge was to create a tool that seamlessly integrates into existing design workflows while providing meaningful assistance with AI interaction. The plugin underwent extensive user testing with professional designers to ensure it meets real-world needs.

Project Impact

PromptlyUX represents a significant contribution to the field of AI-assisted design, demonstrating how thoughtful tool design can bridge the gap between human creativity and artificial intelligence. The research findings from this project provide valuable insights into the future of design workflows and AI integration.
