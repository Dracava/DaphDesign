PromptlyUX

Web Design Apr 30, 2025

Project Overview

PromptlyUX is a Figma plugin designed to support UI/UX designers in crafting effective prompts for AI tools like ChatGPT. It bridges the gap between creative workflows and AI systems by offering structured prompt guidance tailored to the design process.

Key Features

  • AI Chat Interface: A contextual chat powered by a large language model, offering design-specific guidance. Each response includes a clickable follow-up suggestion to streamline iteration.
  • Prompt Guidance System:
    • Getting Started: A 10-question survey that collects project details to help generate relevant prompts.
    • Prompt Library: A categorized collection of prompt examples and UX keywords to inspire users.
    • Prompt Generator: Enter a topic (e.g., "buttons") to instantly generate prompt variations.
    • Saved Prompts: Manage custom prompts for reuse and editing.
  • Interactive UI Tabs: Users can toggle between design stages (e.g., Personas, Style Guide), each with curated prompt templates.
  • Accessibility Tools: Adjustable text sizes and intuitive navigation enhance usability for diverse users.

Project Context

Developed as part of my master's thesis on AI-assisted design workflows, PromptlyUX explores how structured prompting can enhance creativity, efficiency, and AI literacy among designers.

Technologies Used

  • Figma Plugin API
  • React for plugin interface
  • Large Language Model Integration
  • User Research & Testing
